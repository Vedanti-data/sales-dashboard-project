# Sales Performance Dashboard

## About This Project
Analyzed 9,994 rows of retail sales data to uncover
revenue trends, top products, and regional performance
using Python and Power BI.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI Desktop
- Dataset: Kaggle Superstore (9994 rows, 21 columns)

## Key Business Insights
- Technology is highest revenue category ($836,154)
- West region contributes maximum sales ($725,457)
- Q4 is peak sales season every year
- Canon imageCLASS 2200 is top selling product ($61,599)
- Furniture has lowest profit despite high sales

## Project Files
- analysis.py — data cleaning and analysis code
- superstore_cleaned.csv — cleaned dataset
- chart1 to chart5 — visualization images
- sales_dashboard.pbix — Power BI dashboard

## Charts Created
1. Sales by Category — Bar Chart
2. Monthly Sales Trend — Line Chart
3. Sales by Region — Pie Chart
4. Top 10 Products — Horizontal Bar Chart
5. Profit vs Sales — Scatter Plot
